## Exit Criteria

- The primary SDR workflow can be reviewed from queue to detail to Digital
  Worker assignment and progress.
- Gate-failed leads have a clear, draft-free UI state.
- Send-like actions remain explicit and review-gated.
- Frontend validation covers behavior beyond static rendering.
