# Implementation Plan

Signal should be built in narrow, demo-safe phases. Each row below is scoped for
one coding-agent thread using the repo's loop system.

## Phase Index

| Phase | Goal | File |

| Phase 1 - Spec And Baseline | Establish the fresh PRD, local agent guidance, and baseline alignment tasks | `phase-1-spec-and-baseline.md` |

| Phase 3 - SDR Workspace | Build the queue, lead detail, run status, review, and copy/export workflow | `phase-3-sdr-workspace.md` |
| Phase 4 - Evals, Rollout, Demo | Prove scoring, citations, safety, and rollout readiness | `phase-4-evals-rollout-demo.md` |
