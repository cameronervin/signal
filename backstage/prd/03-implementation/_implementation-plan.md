## Production Hardening Boundaries

- V1 remains a human-reviewed assignment/demo build; it does not add live CRM
  integration, production auth, paid providers, or automatic email sending.
- The rollout docs use a two-week shadow period followed by a four-week pilot as
  the default evaluation sequence.
- Production retention, real delivery tooling, and broader observability remain
  explicit hardening decisions before any live sales-org deployment.
