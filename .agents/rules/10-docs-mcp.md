# Documentation Lookup

Use current official docs before writing library-specific code or configuration.

## Preferred Tools

- `context7` for library, framework, SDK, CLI, and cloud-service docs.
- Official project docs when changing OpenAI, FastAPI, Next.js, LangGraph, SQLAlchemy, Pydantic, Tailwind, Jest, Celery, LiteLLM, or provider-specific code.

## Rules

- Verify syntax against the installed or declared dependency versions.
- Prefer primary documentation over blog posts.
- Record version-sensitive assumptions in the implementation summary when they matter.
