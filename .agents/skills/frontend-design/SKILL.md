# Frontend Design

Use this skill for Signal UI work.

## Workflow

1. Read `backstage/design/README.md`.
2. Build reusable primitives before route-specific composition.
3. Use design tokens from CSS variables.
4. Verify desktop-first layouts and single-column detail fallback.
5. Preserve first-class gate-failed, awaiting-review, and empty states.

## Rules

- Use lucide-react icons.
- Keep cards for bounded tools and repeated items.
- Do not add landing pages or marketing hero sections.
- Do not use decorative gradients or blobs.
- Ensure text fits in buttons, table cells, and cards.
