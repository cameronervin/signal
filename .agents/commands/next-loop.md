# Next Loop

Use this when a phase or feature slice is complete and the next build loop needs
to be selected.

1. Read `backstage/prd/03-implementation/_implementation-plan.md`.
2. Check completed code and tests.
3. Identify the smallest next user-visible outcome.
4. Pick one loop from `.agents/loops/`.
5. Write the plan to `.agents/plans/` if the work spans more than one layer.
